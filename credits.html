Website / Software Details

  • Coding was done in Phoenix Code and is hosted on github
    • On ocassion I code on the codeit website for mobile coding + pushing
  • Tested on Firefox at a 1920x1080 resolution
  • Primary fonts used are Free Pixel and Venice from dafont, Dosis and Atkinson Hyperlegible from Google Fonts, and Intel One Mono (which is also what I use to code)
  • Contrast finder is used for any on hand colour contrast checking
  • Word HTML for a WYSIWYG text editor
  • PNGGauntlet, Squoosh and iloveimg for image optimizing
